Getting Started Quickly

Here is a short and straight-forward guide to get started quickly with toMOTko.

  • Set the study languages
  • Import existing glossaries
  • Create your own glossaries
  • Set up and start a quiz

Set the study languages

By default, toMOTko's study language are English and Japanese. If you want to learn different or additional languages, you must specify them in the Preferences like this:

  1. Click the Preferences item in the application menu.
  2. In the Preferences dialog, click the Study Languages... button.
  3. A list of languages will appear where you can check (or uncheck) languages of interest. You must choose at least 2 languages, including your mother tongue. For example, if you're French and want to learn Russian and Chinese, you would check French, Russian, and Chinese languages.
  4. Click Ok to apply the changes.
  5. Set the languages selectors properly. The language selectors are the two fields on the top-left corner of the Glossary Manager. The first one must contain your mother language (or source language) and the second one must contain the language you want to learn (or target language).

Import existing glossaries

There are some glossaries already available on toMOTko's website in the Glossaries section. If you see something that fits your needs there, download the glossary files into your computer. Do not unzip those files.

To import a glossary into toMOTko, do this:

  1. Click the folder into which you want to import the glossary. The very first time you use toMOTko, you will have only one folder named My Glossaries.
  2. Click the Import Data item in the application menu.
  3. Locate a downloaded glossary file and select it.

The content of the glossary file should appear inside the selected folder. Repeat the procedure for all the downloaded glossaries.

Create your own glossaries

If none of the available glossaries interest you. You can create your own glossaries like this:

  1. Click the Red Book button (middle of the left side of the Glossary Manager) to add a new and empty glossary.
  2. To add new words in the glossary, click on it and select the Words tab (on the right side).
  3. Click the White Sheet button (middle of the right side) to insert a new word.
  4. Enter the word's data and click Ok.
  5. Repeat as needed.

Set up and start a quiz

To set up a quiz, you must specify which words the quiz will ask. To do that:

  1. Check all the words you want
  2. Check also the parent glossaries
  3. Check also the parent folders, if any.
  4. To start the quiz, click the Start Quiz item in the application menu.
  5. To gradually reveal information of the word being asked, click the Eye button as many times as needed.
  6. To signify a right answer, so that toMOTko stops asking the word, click the Green Circle button. To signify a wrong answer, so that toMOTko keeps asking the word, click the Red X button. By default, a word is repeated at least 2 times. You can increase or decrease this value in the Preferences.
